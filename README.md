## Требования

- docker
- docker compose v2

## Сборка

```bash
sudo docker compose up --build
```

## Запуск
Клиентское приложение доступно на порту 80, зайти на него можно из браузера по адресу
`http://localhost` или 'http://127.0.0.1'
